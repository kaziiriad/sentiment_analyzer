API Overview

POST /api/analyze/

Description: Analyze Text Sentiment Using a Pre-trained ML Model

Request Body:

{
    "text": "Text to be analyzed"
}

Example response:

{
    "sentiment": "positive/negative/neutral"
}

GET /api/analyze/all/

Description: Retrieve All The Analyzed Data From The Database.

Example response:

{
    "text" : "text1",
    "sentiment": "positive"
},
{
    "text" : "text1",
    "sentiment": "positive"
},
{
    "text" : "text3",
    "sentiment": "negative"
},

GET /docs/

Description: API Documentation - SwaggerUI.
